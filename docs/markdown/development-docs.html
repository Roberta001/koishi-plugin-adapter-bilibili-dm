文档开发指南 ​

本页面介绍如何参与本文档站点的开发和维护。

环境准备 ​

系统要求 ​

  • Node.js >= 22.0.0
  • Yarn 或 npm
  • Git
  • 基本的 Markdown 和 VitePress 知识

快速开始 ​

  1. 克隆文档仓库
bash
git clone https://github.com/Roberta001/koishi-plugin-adapter-bilibili-dm.git
  1. 切换到文档分支
bash
cd koishi-plugin-adapter-bilibili-dm
git checkout docs
  1. 安装依赖
bash
yarn install
  1. 启动开发服务器
bash
yarn dev
  1. 构建文档
bash
yarn build
